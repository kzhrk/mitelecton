# MiteTray <img src="./resources/icon.png" height="24">

システムトレーに常駐するMiterasクライアント

## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
```

### Build

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac
```
